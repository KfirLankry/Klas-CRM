About KLAS

What is KLAS?
K.L.A.S is an online Customer Relationship Management (CRM) single page End-to-End application developed for car rental companys. The system covers both customer and day to day order managments.
Project Features: CRUD Operations, Database Analysis, User Authentiction, Custom Intefaces for Data Types, Guarded Routes.
Tools Used: Angular, Typescript, Firebase Hosting, Firestore database.
K.L.A.S stands for: Kfir Lankry & Aviv Shleyfman, Two full-stack web developers that collaborated to create this web as part of their Angular summary project.
