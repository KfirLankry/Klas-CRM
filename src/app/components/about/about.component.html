About KLAS

What is KLAS?
K.L.A.S is an online CRM system for a fake car rental company.
Covers both customer and day to day order managment
K.L.A.S stands for: Kfir Lankry & Aviv Shleyfman
Two full-stack web developers that collaborated to create this project
Project features:
  • CRUD Operations
  • Database Analysis
  • User Authentiction
  • Custom Intefaces for Data Types
  • Guarded Routes
Tools used:
  • Angular
  • Typescript
  • Firebase Hosting
  • Firestore database
